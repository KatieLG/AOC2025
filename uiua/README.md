# Uiua Solutions

Advent of Code solutions written in [Uiua](https://www.uiua.org/) version 0.17.2.

## Running Solutions

| Command             | Description                        |
| ------------------- | ---------------------------------- |
| `make run`          | Run today's solution               |
| `make run day=01`   | Run solution for specified day     |
| `make test`         | Run tests                          |
| `make setup day=01` | Create template solution for a day |
